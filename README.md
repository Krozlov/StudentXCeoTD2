- Styling: Tailwind CSS

## Installation & Setup
1. Clone the repository
   ```
   git clone https://github.com/Krozlov/StudentXCeoTD2
   ```
2. Install dependencies
   ```
   npm install
   ```
3. Environment configuration
   Create a `.env.local` file in the root directiory and add your API keys:
   ```
   GEMINI_API_KEY=enter_your_api
   ```
4. Run development server
   ```
   npm run dev
   ```
